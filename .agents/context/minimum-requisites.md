# Requisitos de evaluación de calidad obligatorios

- **Arquitectura y Patrones:** Implementa un patrón claro (Modelo-Vista-ViewModel, Clean Architecture). Desacoplamiento total entre lógica y vista.
- **Código y Sintaxis Swift:** Uso avanzado de Swift: Optionals (if let/guard), Enums con valores asociados, Protocolos y Extensiones.
- **Interfaz de Usuario (UI/UX):** Diseño profesional siguiendo las Human Interface Guidelines de Apple. Uso fluido de Auto Layout o SwiftUI.
- **Gestión de Datos y API:** Integración perfecta con APIs REST o persistencia local (CoreData/SwiftData). Manejo eficiente de concurrencia.
- **Rendimiento y Memoria:** Sin fugas de memoria (Memory Leaks). Uso correcto de weak y unowned. Carga rápida de assets.
