# Contexto general del proyecto

## Alcance
Este repositorio contiene un catalogo de 10 apps iOS planeadas para desarrollo progresivo.

## Estado actual
- El proyecto base existe en Xcode.
- De momento hay una app activa como objetivo inmediato, pero la documentacion debe seguir siendo util para cualquiera de las 10 apps.

## Normas obligatorias
- MVVM es obligatorio.
- Clean Code es obligatorio.
- Clean Architecture es obligatoria.
- UIKit es la base principal de implementacion.
- Se prioriza el uso de Table View, Tab Bar Controller, NotificationCenter, UserDefaults, UIView y Core Animation segun el caso de uso.

## Criterios de trabajo
1. Mantener el contexto general sin casarse con una sola app.
2. Ajustar la implementacion a la app activa del momento.
3. No perder compatibilidad con el proyecto Xcode ya creado.
