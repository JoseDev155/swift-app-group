# Objetivo del proyecto

Crear y evolucionar un catalogo de apps iOS.

Se usara Swift, UIKit, Table View y, cuando el flujo lo requiera, Tab Bar Controller, NotificationCenter, UserDefaults, UIView y Core Animation.

Cada app debe resolver su problema con una experiencia funcional, limpia y minimalista, siguiendo las Human Interface Guidelines de Apple.

La base del proyecto ya esta creada con la plantilla por defecto de Xcode. El objetivo es transformar esa base en implementaciones consistentes, modulares y mantenibles.
